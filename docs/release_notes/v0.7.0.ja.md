# xer C++ Utility Library v0.7.0 リリースノート

xer C++ Utility Library v0.7.0 では、ライブラリの数値処理まわりを拡張しました。このリリースでは、実用的な実数数学、記述統計、複素数の方程式ヘルパーのための公開ヘッダを追加し、math、matrix、cyclic、image API の角度扱いを τrad 単位に整理しています。

また、v0.7.0 の API に合わせて、README、公開ヘッダ一覧、リファレンスマニュアル断片、日本語リファレンス断片、用語表、方針文書を更新しました。

## ハイライト

- `<xer/math.h>` を追加し、軽量な実数数学、小さなベクトル、極座標、三角関数、ベクトルヘルパー、方程式ヘルパーを提供
- `<xer/statistics.h>` を追加し、算術範囲に対する記述統計関数を提供
- `<xer/complex.h>` を追加し、複素根を返す方程式ヘルパーを提供
- xer の角度 API を τrad 単位に統一
- `cyclic` の角度方針に度数法・ラジアンとの変換ヘルパーを追加
- 行列の回転ヘルパーを cyclic τrad 角に整理
- 画像の円弧 API とドキュメントを τrad 角に整理
- README、公開ヘッダ一覧、リファレンスマニュアル断片、翻訳用語表、方針文書を v0.7.0 API に合わせて更新

## 新しい公開ヘッダ

### `<xer/math.h>`

`<xer/math.h>` は、xer における軽量な実数数学ヘルパーを提供します。対象は実用的な数式や小規模な幾何計算であり、本格的な数値解析ライブラリを目指すものではありません。

主な追加内容は次のとおりです。

- 2次元、3次元、4次元の小さなベクトルを表す `vec<T, N>`
- cyclic τrad 角を使う極座標 `polar<T, 2>`
- τrad スカラー値または `cyclic<T>` 値を使う三角関数
- τrad 値を返す逆三角関数
- 極座標・直交座標の変換ヘルパー
- `dot`、`length`、`distance`、`normalize`、`angle`、`rotate`、`cross` などのベクトルヘルパー
- 三角形の面積を求める `heron`
- 実根を求める `quadratic` と `cubic`

`<xer/math.h>` の角度は τrad 単位です。この規約では、`1` が1回転、`0.25` が直角、`0.5` が半回転を表します。

### `<xer/statistics.h>`

`<xer/statistics.h>` は、算術範囲に対する小さな記述統計ユーティリティ関数を提供します。

提供する関数は次のとおりです。

- `sum`
- `product`
- `mean`
- `geometric_mean`
- `harmonic_mean`
- `median`
- `quantile`
- `percentile`
- `mode`
- `variance`
- `sample_variance`
- `stddev`
- `sample_stddev`

多くの関数は `xer::result<double>` を返します。これにより、空範囲や不正なパラメータなどの不正入力を明示的に報告できます。`mode` は、複数の値が同じ最高頻度を持つ場合があるため、`xer::result<std::vector<double>>` を返します。

`mode` は任意の許容幅も受け取れます。これにより、完全一致だけでは扱いにくい浮動小数点数データにも使いやすくしています。

### `<xer/complex.h>`

`<xer/complex.h>` は、一部の方程式ヘルパーに対する複素数版を提供します。

提供する関数は次のとおりです。

- `cquadratic`
- `ccubic`

これらの関数は実係数を受け取り、重複度を含む複素根を返します。実数ヘルパーだけを使うコードが複素数機能を含めずに済むよう、`<xer/math.h>` とは分離しています。

## 角度方針と τrad 統一

v0.7.0 では、関連する xer API の角度扱いを τrad 単位に整理しました。

τrad 単位では次のように表します。

- `1` は1回転
- `0.5` は半回転
- `0.25` は直角

この方針は、次の API に一貫して適用されます。

- `<xer/math.h>` の三角関数
- `<xer/cyclic.h>` の cyclic 角変換ヘルパー
- `<xer/matrix.h>` の回転ヘルパー
- `<xer/image.h>` の円弧描画 API

度数法やラジアンを使う外部 API とやり取りするために、`from_degree`、`to_degree`、`from_rad`、`to_rad` などの変換ヘルパーを提供します。

## 行列と画像の更新

`<xer/matrix.h>` では、回転ヘルパーを cyclic τrad ベースの操作として文書化しました。これには、2次元回転と、x・y・z 軸まわりの3次元回転が含まれます。

`<xer/image.h>` では、円弧・楕円弧の角度を τrad 単位として文書化しました。これにより、画像描画も xer 全体の角度方針と揃い、単位混在によるミスを減らしやすくなっています。

## 互換性メモ

v0.7.0 は新しい公開ヘッダと関数を追加するリリースです。既存の v0.6.0 API は、文書化されている v0.x の安定性方針の範囲にあります。

実用上もっとも重要な互換性上の注意点は角度単位です。新しく文書化された math、matrix、cyclic、image の角度 API は、明示的に別記されていない限り τrad を使います。度数法やラジアンを使う API とやり取りする場合は、提供されている変換ヘルパーを使ってください。
